## 👾 What Is This?

A galaxy-inspired React UI experiment with:
- A floating, blinking Spline robot 🤖
- A sparkly custom cursor with trailing stars 🌟
- Cosmic gradient backgrounds 💜
- Floating animated text
- Absolutely no sleep involved 😵‍💫

- 🎨 CSS animations
- 🛠️ Spline 3D Editor
- 💫 Netlify (for hosting)
